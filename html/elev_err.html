ERROR EXAMPLES:

Sources of error

Collectors error
Wrong initial determination of elevation; wrong county; rounding error
Primary transcription error
Correct or missing in fieldbook, wrong on label; values, units
Secondary transcription error
Correct on label, wrong in database
Interpretation error
Data missing on label, incorrectly inferred in database
Data maintenance error
Systematic alteration of values; failure to update database
Temporal error
Changes in map; changes in territory
Multiple Error
(sometimes compensating, and therefore silent)
Analysis error
Incorrect maximum elevation; unexpected data format
JEPS records where elevation is more than 500 feet higher than maximum elevation in county.
